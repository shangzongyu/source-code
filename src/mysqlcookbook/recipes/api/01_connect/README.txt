This directory contains recipes showing how to connect to the MySQL
server. The languages demonstrated are:

Java:
  Connect.java

PHP:
  connect.php

Perl:
  connect.pl

Python:
  connect.py

Ruby:
  connect.rb

Go:
  connect.go

The "connect2" and "connect3" scripts in this directory show
alternative methods for connecting to the server, generally by
specifying a TCP/IP port number or Unix domain socket file value
other than the default.

The "connect4" scripts show how to specify the character set at connect time.
